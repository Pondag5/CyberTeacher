# 🛡️ Курс "Кибербезопасность с нуля"

## Неделя 1: Основы
**Цель:** Понять базовые концепции и терминологию

### Темы:
- Что такое кибербезопасность
- Типы угроз (вирусы, хакеры, социальная инженерия)
- Основные термины: CVE, эксплойт, уязвимость, вектор атаки
- Модели угроз (MITRE ATT&CK)

### Книги для изучения:
- Blue Team Field Manual (глава 1-2)
- RTFM (глава 1)
- Cybersecurity for Business

### Практика:
✅ Настрой 2FA на важных аккаунтах
✅ Создай менеджер паролей
✅ Проверь свои данные на утечки (haveibeenpwned.com)

## Неделя 2: Основы сетей
**Цель:** Понимать как работает интернет

### Темы:
- Модель OSI (7 слоёв)
- IP адреса, подсети
- DNS — как работает преобразование имён
- TCP/UDP — протоколы передачи данных
- Порты и сервисы (HTTP, HTTPS, SSH, FTP)

### Книги для изучения:
- CompTIA Network Lab Manual
- IPv4_Subnetting.pdf
- common_ports.pdf

### Практика:
✅ Изучи Wireshark — посмотри трафик своего ПК
✅ Настрой Wireshark_Display_Filters
✅ Изучи базовые команды: ping, traceroute, nslookup

### Книги для изучения:
