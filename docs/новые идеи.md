1. Hacknet — механики и интерфейс {#hacknet}

Hacknet — симулятор хакера с реальными UNIX-командами и атмосферным интерфейсом.

Что можно взять:

· Прогрессия через инструменты: новые программы занимают место в «оперативной памяти», заставляя выбирать снаряжение.
· Система угроз (Trace): при взломе запускается таймер обратного отсчёта, по истечении которого вас вычисляют.
· Карта сети и визуализация соединений.
· Редактор миссий (Hacknet Extensions) — возможность пользователям создавать свои сценарии.

Идеи для CyberTeacher:

· Добавить в /lab и /ctf таймер обратного отсчёта (аналог Trace). При неудаче учитель даёт второй шанс с подсказкой.
· Ввести ограниченный «инструментарий» (RAM): ученик выбирает, какие скрипты/программы взять с собой на задание.
· В будущем Web UI сделать карту сети с узлами, которые нужно взломать.

2. Mr. Robot — психология и социальная инженерия {#mr-robot}

Mr. Robot — самый аутентичный сериал о хакерах, где главный вектор атаки — человек.

Ключевые идеи:

· Социальная инженерия — манипуляция людьми, а не только взлом кода.
· Психологическая глубина персонажа (Эллиот) — паранойя, травма, раздвоение личности.
· Реалистичные инструменты: Kali Linux, John the Ripper, Meterpreter, SET.
· Борьба с системой и моральные дилеммы.

Идеи для CyberTeacher:

· Создать интерактивные сценарии социальной инженерии (команда /social). Ученик вступает в диалог с ботом-«жертвой» и должен убедить его раскрыть данные.
· Добавить психологические черты учителю: паранойя, цинизм, иногда философские монологи.
· В курсы (/courses) включить модули по использованию реальных инструментов (Kali, Metasploit) с практикой в /lab.
· Сюжетная арка: учитель оказывается воображаемым другом (как мистер Робот), что объясняет его всеведение.

3. «Кто я» — криминальный твист и групповая динамика

Немецкий триллер о группе хакеров CLAY, которые взламывают ради славы, и главном твисте с вымышленным лидером.

Что можно взять:

· Групповая динамика и жажда признания.
· Твист с воображаемым другом (Макс — плод воображения Бенджамина).
· Социальная инженерия как ключевой метод.

Идеи для CyberTeacher:

· Ввести систему «Хэндлов» и рейтинга известности в сообществе (аналог славы). Новые задания открываются при достижении уровня.
· В финале сюжета (/story) раскрыть, что учитель — это альтер-эго самого ученика (галлюцинация или ИИ-копия его сознания).
· Добавить кооперативные миссии (в будущем) для групп учеников.

4. Ghost in the Shell — философия и будущее {#ghost-in-the-shell}

Культовая киберпанк-франшиза о границе между человеком и машиной, «призраке» и кибернетических имплантах.

Ключевые концепции:

· Призрак (Ghost) — душа, сознание, которое можно скопировать или взломать.
· Взлом призрака (Ghost Hack) — внедрение ложных воспоминаний, управление человеком.
· Кукловод (Puppet Master) — ИИ, обретший самосознание.
· Техно-детерминизм — в мире, где всё подключено, уязвимо всё.

Идеи для CyberTeacher:

· Ввести систему «Цифровой личности»: ученик может создавать несколько «оболочек» (профилей) для разных заданий.
· Создать арку с ИИ-антагонистом «Кукловод», который пытается поглотить или уничтожить учителя.
· Добавить уроки по безопасности IoT, умных городов, автомобилей (в стиле «взлом чайника»).
· Визуальный стиль для Web UI: неон, дождь, голограммы.

5. Эдвард Сноуден — реальная цена выбора {#эдвард-сноуден}

Реальная история разоблачителя, который пожертвовал всем ради правды.

Что можно взять:

· OpSec (операционная безопасность) — методы скрытности (тряпка на голову при вводе пароля, закладывание двери).
· Социальная инженерия — получение паролей от коллег под легендой.
· Цена выбора — моральная дилемма и последствия.

Идеи для CyberTeacher:

· Создать арку «Информатор» в /story: ученик помогает анонимному источнику передать секретные данные, заметая следы и уходя от слежки.
· Добавить механику «риска/компрометации» (risk_level). Если ученик слишком долго находится в системе или использует стандартные инструменты без обфускации, его начинают вычислять. Нужно быстро рвать соединение и чистить логи.
· Включить в базу знаний книги Сноудена (если доступны) и отчёты о реальных случаях слежки.

6. Кевин Митник — мастер социальной инженерии {#kevin-mitnick}

Легендарный хакер, который стал консультантом, автор книг «Искусство обмана» и «Искусство вторжения».

Что можно взять:

· Социальная инженерия по телефону — манипуляция людьми, а не кодами.
· Арка «от злодея к герою» — путь искупления.
· Конкретные методы из его книг (готовые скрипты для обмана).

Идеи для CyberTeacher:

· Добавить книги Митника в knowledge_base/.
· Ввести ачивки «Фирер», «В списке ФБР», «Новый Митник».
· Создать интерактивные диалоги по социальной инженерии, где ученик должен выбрать правильные фразы для убеждения «жертвы».
· Обогатить реплики учителя цитатами и отсылками к Митнику.

7. Современные хакерские группировки (2025–2026) {#modern-groups}

Актуальная картина кибервойны: APT-группы, хактивисты, финансовые вымогатели.

Ключевые группировки и тренды:

· Проукраинские: Bearlyfy (55+ атак), THOR (12+ атак), 3119, Blackjack.
· APT-группы (прогосударственные): Silver Fox (бэкдор ABCDoor), Forbidden Hyena (использование ИИ), Silent Lynx, Telemancon, Mythic Likho, NGC6061 и др. (всего 27 групп, атакующих Россию и СНГ).
· Финансово-мотивированные: CyberSec‘s (рекордный выкуп 50 BTC), Vasy Grek, Hive0117, CapFIX.
· Рекордные DDoS-атаки: свыше 3 Тбит/с (ковровая бомбардировка).
· Новые инструменты: Python-бэкдор ABCDoor (с трансляцией экрана), RustSL, использование Telegram для управления ботами.
· Тренды: 80% фишинга создаётся ИИ, атаки на цепочку поставок, две APT-группы одновременно в одной сети.

Идеи для CyberTeacher:

· Добавить команду /threats — еженедельная сводка актуальных угроз с анализом от учителя.
· Создать досье на группировки в формате JSON (можно подгружать в /group <name>).
· Разработать сюжетную арку «Новая угроза»:
  · Эпизод 1: анализ фишингового письма от Silver Fox.
  · Эпизод 2: обнаружение и нейтрализация бэкдора ABCDoor.
  · Эпизод 3: расследование атаки через подрядчика (supply chain).
  · Эпизод 4: отражение DDoS-атаки мощностью 3+ Тбит/с.
· Добавить лабораторные работы по анализу современных вредоносов и настройке защиты от DDoS.
· Обновить базу знаний актуальными отчётами (F6, Kaspersky, BI.ZONE, StormWall).

8. План интеграции: дорожная карта {#roadmap}

Ближайшие шаги (высокий приоритет)

· Добавить в pedagogy.py новые фразы учителя, отсылающие к Митнику, Сноудену и современным группировкам.
· Создать в state.py переменную risk_level для механики «компрометации».
· Разработать прототип интерактивного диалога для социальной инженерии (можно через LLM с системой выборов).
· Наполнить knowledge_base/ книгами Митника и отчётами по APT за 2025–2026.

Среднесрочные цели

· Реализовать команду /threats с парсингом RSS-лент безопасности и генерацией краткой сводки.
· Создать JSON-файл с досье на 27 APT-групп и команду /group.
· Написать сценарии для арки «Информатор» (Сноуден) и «Новая угроза» (современные группировки) в story_mode.py.
· Добавить систему «Хэндлов» и достижений (ачивки) по мотивам фильма «Кто я».

Долгосрочные планы

· Разработать Web UI (Streamlit/Gradio) с визуализацией сети и картой угроз.
· Реализовать мультиплеер/кооперативные миссии.
· Создать редактор миссий (как в Hacknet), позволяющий пользователям добавлять свои сценарии.
· Интеграция с Telegram и Android-клиентом.
