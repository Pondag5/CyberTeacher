# CyberTeacher - Сводка проекта

## Описание
CLI-приложение для обучения кибербезопасности с LLM-учителем.

## Технологии
- Python 3.14
- LLM: VL Studio (Qwen2.5-3B) / Ollama
- RAG: FAISS + sentence-transformers
- UI: Rich
- DB: SQLite
- Docker лаборатории

## Структура
```
main.py           - главный цикл
handlers.py       - команды
state.py          - состояние
config.py         - настройки
knowledge.py      - база знаний (FAISS)
memory.py         - SQLite
story_mode.py     - 20 эпизодов с флагами
courses.py        - курсы
practice.py       - Docker лабы
```
